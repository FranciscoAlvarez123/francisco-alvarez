# Trabajo Práctico 01 - Productos que utilizan Machine Learning

## Introducción

El Machine Learning (ML) es una rama de la inteligencia artificial que permite a los sistemas aprender a partir de datos para realizar predicciones o tomar decisiones sin ser programados explícitamente.

En este trabajo se investigan cinco productos o servicios reales que utilizan Machine Learning, explicando el problema que resuelven, el tipo de aprendizaje utilizado y los datos necesarios para su funcionamiento.

---

# 1. Netflix - Sistema de recomendación

## Problema que resuelve

Netflix utiliza machine learning para recomendar películas y series a los usuarios según sus gustos y comportamiento dentro de la plataforma.

Esto ayuda a que los usuarios encuentren contenido relevante rápidamente y aumenta el tiempo que pasan en la plataforma.

## Tipo de Machine Learning

**Aprendizaje supervisado y no supervisado**

Se utilizan modelos que analizan patrones en los datos de los usuarios y comparan gustos entre usuarios similares (collaborative filtering).

## Por qué es ese tipo

El sistema aprende a partir de datos históricos de visualización y comportamiento para predecir qué contenido podría gustarle a cada usuario.

## Datos que necesita

- Historial de películas y series vistas
- Calificaciones o interacciones
- Tiempo de visualización
- Género del contenido
- Preferencias de usuarios similares

---

# 2. Google Maps - Predicción de tráfico

## Problema que resuelve

Google Maps utiliza machine learning para predecir el tráfico y calcular el tiempo estimado de llegada (ETA).

Esto permite sugerir las rutas más rápidas a los usuarios.

## Tipo de Machine Learning

**Aprendizaje supervisado**

## Por qué es ese tipo

El modelo se entrena con grandes cantidades de datos históricos de tráfico y aprende a predecir cuánto tardará un viaje según las condiciones actuales.

## Datos que necesita

- Ubicación de los usuarios
- Velocidad de vehículos
- Datos históricos de tráfico
- Condiciones de las rutas
- Hora del día

---

# 3. Mercado Libre - Detección de fraude (Argentina)

## Problema que resuelve

Mercado Libre utiliza machine learning para detectar transacciones fraudulentas en su plataforma de comercio electrónico.

Esto ayuda a prevenir estafas y proteger tanto a compradores como vendedores.

## Tipo de Machine Learning

**Aprendizaje supervisado**

## Por qué es ese tipo

El sistema se entrena con datos de transacciones pasadas que fueron clasificadas como fraudulentas o legítimas para poder predecir nuevas situaciones.

## Datos que necesita

- Historial de transacciones
- Datos de pagos
- Ubicación del usuario
- Dispositivo utilizado
- Comportamiento del comprador

---

# 4. Waymo - Autos autónomos

## Problema que resuelve

Waymo desarrolla autos autónomos que pueden conducir sin intervención humana utilizando inteligencia artificial y machine learning.

El objetivo es mejorar la seguridad vial y automatizar el transporte.

## Tipo de Machine Learning

**Aprendizaje por refuerzo y deep learning**

## Por qué es ese tipo

Los vehículos aprenden a tomar decisiones en la conducción a partir de simulaciones y experiencias reales, optimizando sus acciones para conducir de forma segura.

## Datos que necesita

- Imágenes de cámaras
- Datos de sensores
- Radar
- Lidar
- Información del entorno y tráfico

---

# 5. Spotify - Recomendación de música

## Problema que resuelve

Spotify utiliza machine learning para recomendar canciones, artistas y playlists personalizadas a los usuarios.

Esto mejora la experiencia del usuario y facilita descubrir nueva música.

## Tipo de Machine Learning

**Aprendizaje no supervisado y supervisado**

## Por qué es ese tipo

El sistema analiza patrones de escucha entre usuarios para encontrar similitudes y recomendar música que probablemente le guste al usuario.

## Datos que necesita

- Historial de canciones escuchadas
- Playlists creadas
- Artistas favoritos
- Interacciones (likes, skips)
- Datos de otros usuarios con gustos similares

---

# Conclusión

El Machine Learning se utiliza ampliamente en productos digitales modernos para analizar grandes cantidades de datos y mejorar la experiencia del usuario. Empresas como Netflix, Google, Mercado Libre y Spotify utilizan estos sistemas para hacer recomendaciones, prevenir fraudes y optimizar servicios. Esto demuestra el impacto que tiene la inteligencia artificial en la tecnología actual.
